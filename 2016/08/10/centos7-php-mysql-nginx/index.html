安装Centos7

  • 下载iso文件
    因为系统是64位，所以直接下载64位iso文件 点此下载。

  • 使用Virtualbox安装好，步骤略…

  • 安装后配置

    • 查看当前系统环境

    • 设置网络

      默认竟然不带ifconfig命令了，通过nmtui设置好网络后，使用yum install net-tools可以安装ifconfig，再输入ifconfig就可以了。

      详细说明参考http://www.krizna.com/centos/setup-network-centos-7/

  • 重启网络

  • 启动sshd，方便putty等ssh远程连接

  • 安装相关辅助工具

    1. 安装 vim

    2. 安装 wget

    3. 安装 tree 以树型方式展示目录结构

    4. 安装lsof （list open files）是一个列出当前系统打开文件的工具

    5. 安装gcc gcc+ (编译安装软件时用到)

    6. 安装rz/sz 上传下载工具 (lrzsz是一个搭配SecureCRT使用的上传下载工具)
